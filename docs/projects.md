## Project 2
title: SHIELD: Streaming, Hierarchical, Interpretable Eye-Risk Detection using Lens Data
funding: Pending
role: Principal Investigator
value: $--
years: 2026-
image: assets/img/research/02.png
link: research-p2.html
desc: This project develops SHIELD, a clinically grounded framework for intraoperative monitoring that uses Triggerfish contact-lens data together with cardiovascular signals to detect ocular-risk-relevant abnormalities in real time. The framework integrates artifact-aware representation learning, state-conditioned multimodal reconstruction, and streaming hierarchical anomaly detection to provide robust and interpretable decision support during surgery.
